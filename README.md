Project 4 – E-Commerce Order Analysis & Visualization

Overview

This project analyzes e-commerce order data to uncover key business insights related to sales performance, customer activity, product demand, order fulfillment, and payment behavior.

Using Excel-based data analysis and visualization techniques, the project transforms raw transactional data into actionable business intelligence that can support decision-making.



Dataset Information

The dataset contains 1,200 customer orders and includes information such as:

* Order ID
* Customer ID
* Product
* Quantity
* Unit Price
* Total Price
* Payment Method
* Order Status
* Referral Source
* Coupon Usage
* Shipping Information



Project Objectives

* Analyze overall sales performance
* Evaluate customer purchasing behavior
* Understand order fulfillment trends
* Assess payment method preferences
* Identify opportunities for operational improvement
* Present findings through data visualization

Order Status Analysis

The dataset includes five order statuses:

* Delivered
* Shipped
* Pending
* Returned
* Cancelled

Insights

* Delivered orders represent completed sales.
* Returned and cancelled orders highlight potential operational challenges.
* Pending and shipped orders provide visibility into the fulfillment pipeline.



Payment Method Analysis

The project examines customer payment preferences across multiple payment methods, enabling businesses to:

* Understand customer purchasing habits
* Optimize payment processing strategies
* Improve customer experience



Data Visualizations

The dashboard and visualizations focus on:

* Revenue performance
* Order status distribution
* Payment method breakdown
* Product sales analysis
* Customer purchasing trends
* Operational performance metrics



Business Insights

Key findings include:

* Strong overall revenue generation from 1,200 transactions.
* A high average order value indicates healthy customer spending.
* Nearly every order is associated with a unique customer, suggesting a broad customer base.
* Order status analysis provides insight into fulfillment efficiency and customer satisfaction.
* Payment method trends help identify customer preferences.
